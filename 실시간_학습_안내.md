# 실시간 학습 기능 안내

## 기능 개요

테스트 모드에서 실행 시, AI가 자동으로 게시글의 본문과 댓글을 수집하여 학습하는 기능입니다.

## 작동 방식

### 1. 자동 댓글 수집
- 각 게시글 처리 시 해당 게시글의 실제 댓글을 자동으로 수집
- 수집한 댓글을 즉시 학습 데이터에 추가
- AI 생성기를 재초기화하여 새로운 댓글을 반영

### 2. 상세 로그 기록
- `learning_log.txt` 파일에 다음 정보를 기록:
  - 게시글 URL
  - 게시글 제목
  - 게시글 본문 (처음 500자)
  - 실제 댓글들 (전체)
  - AI가 생성한 댓글

### 3. 점진적 학습
- 게시글을 처리할수록 더 많은 댓글 데이터 수집
- 수집한 댓글이 많아질수록 AI가 더 자연스러운 댓글 생성
- 처리한 게시글 수와 학습한 댓글 수를 실시간으로 표시

## 사용 방법

1. **테스트 모드 체크**: 웹 GUI에서 "테스트 모드" 체크박스 선택
2. **매크로 시작**: 설정 입력 후 "시작" 버튼 클릭
3. **자동 학습**: 게시글을 처리하면서 자동으로 댓글 수집 및 학습
4. **로그 확인**: `learning_log.txt` 파일에서 상세 내용 확인

## 로그 파일 형식

```
================================================================================
게시글 처리 시간: 2025-12-02 10:30:15
================================================================================
게시글 URL: https://oncapan.com/bbs/free/12345
제목: 게시글 제목
--------------------------------------------------------------------------------
게시글 본문:
게시글 내용...
--------------------------------------------------------------------------------
실제 댓글 수: 5개
실제 댓글들:
  1. 첫 번째 댓글
  2. 두 번째 댓글
  3. 세 번째 댓글
  4. 네 번째 댓글
  5. 다섯 번째 댓글
--------------------------------------------------------------------------------
AI 생성 댓글: AI가 생성한 댓글
================================================================================
```

## 학습 효과

- **처음**: 수집된 댓글이 없어 기본 모드로 댓글 생성
- **중간**: 일부 댓글 수집 후 해당 스타일을 참고하여 생성
- **나중**: 많은 댓글 수집 후 실제 사용자와 유사한 스타일로 생성

## 주의사항

- 테스트 모드에서만 자동 수집이 활성화됩니다
- 실제 댓글 작성 모드에서는 자동 수집이 비활성화됩니다 (수동 수집 사용)
- 로그 파일은 계속 쌓이므로 주기적으로 확인하세요
