# 댓글 스타일 개선 방안

## 현재 문제점
- AI 댓글이 너무 완벽하고 정형화됨
- 실제 사용자 댓글과 차이가 남

## 개선 방안

### 1. 댓글 길이
- **현재**: 50자 이내
- **개선**: 10-30자로 더 짧게
- 실제 사용자들은 매우 짧게 댓글을 남김

### 2. 구어체 사용
- "~요", "~네요", "~어요" 같은 구어체 어미
- "좋아요~", "맞아요!", "그렇네요" 같은 느낌

### 3. 약간의 불완전함 허용
- 완벽한 문장보다는 약간 불완전한 느낌
- 띄어쓰기 실수 허용
- 약간의 오타 허용

### 4. 감정 표현 다양화
- 공감: "공감합니다", "맞아요", "그렇네요"
- 의견: "좋은 정보네요", "도움됐어요"
- 질문: "어떻게 하면 되나요?", "진짜요?"
- 단순 반응: "오", "와", "좋아요", "응", "ㅇㅇ"

### 5. 이모티콘 최소화
- 가끔 "~" 정도만 사용
- 이모티콘은 거의 사용하지 않음

### 6. Temperature 조정
- 현재: 0.8
- 개선: 1.0 (더 다양한 표현)

### 7. 추가 개선 아이디어
- 실제 댓글 데이터를 학습에 활용 (옵션)
- 댓글 패턴 다양화
- 게시글 내용에 따른 댓글 스타일 조정
