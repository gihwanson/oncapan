========================================
온카판 자동 댓글 매크로 실행 방법
========================================

1. "실행.bat" 파일을 더블클릭하여 실행하세요.

2. 프로그램이 자동으로:
   - 필수 패키지 확인
   - 패키지 설치 (필요시)
   - GUI 창 실행

3. GUI 창이 열리면:
   - 로그인 정보 입력
   - OpenAI API 키 입력
   - 설정 저장
   - 테스트 모드로 먼저 테스트 (권장)
   - 시작 버튼 클릭

========================================
주의사항
========================================

- 처음 실행 시 패키지 설치에 시간이 걸릴 수 있습니다.
- GUI 창이 열리지 않으면 error.log 파일을 확인하세요.
- 테스트 모드로 먼저 실행하여 정상 작동을 확인하세요.
