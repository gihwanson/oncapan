온카판 자동 댓글 매크로 - 사용 설명서
========================================

1. 프로그램 실행
   - "온카판_자동댓글_매크로.exe" 파일을 더블클릭하여 실행합니다.

2. 초기 설정
   - 프로그램을 처음 실행하면 설정을 입력해야 합니다.
   - 로그인 정보: 온카판 아이디와 비밀번호를 입력하세요.
   - OpenAI API 키: OpenAI API 키를 입력하세요.
   - 댓글 작성 시간 설정: 대기 시간을 설정하세요.
   - "설정 저장" 버튼을 클릭하여 설정을 저장하세요.

3. 테스트 모드
   - "테스트 모드" 체크박스를 선택하면 실제로 댓글을 작성하지 않고 테스트만 합니다.
   - 처음 사용할 때는 테스트 모드로 실행해보는 것을 권장합니다.

4. 매크로 시작
   - 모든 설정을 완료한 후 "시작" 버튼을 클릭합니다.
   - 프로그램이 자동으로 게시글을 찾아 댓글을 작성합니다.
   - 실행 로그에서 진행 상황을 확인할 수 있습니다.

5. 매크로 중지
   - "중지" 버튼을 클릭하여 매크로를 중지할 수 있습니다.

6. 실행 로그
   - 실행 로그에는 다음 정보가 표시됩니다:
     * 게시물 제목
     * 게시물 본문
     * 댓글 목록
     * AI가 작성한 댓글

주의사항:
- OpenAI API 키가 필요합니다.
- 인터넷 연결이 필요합니다.
- Chrome 브라우저가 설치되어 있어야 합니다 (Selenium 사용).
- 프로그램을 종료하려면 "중지" 버튼을 클릭하거나 창을 닫으세요.

문제 해결:
- 프로그램이 실행되지 않으면 Windows Defender나 백신 프로그램에서 차단했을 수 있습니다.
- Chrome 브라우저가 없으면 자동으로 다운로드되지만, 시간이 걸릴 수 있습니다.
