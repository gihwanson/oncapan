# 테스트 모드 EXE 빌드 안내

## 📋 개요

이 스크립트는 타사용자에게 전달할 수 있는 **standalone exe 파일**을 생성합니다.

**중요**: 타사용자는 Python이나 다른 개발 프로그램이 필요 없습니다!
- 개발자(본인)만 빌드 시 Python이 필요합니다
- 타사용자는 생성된 exe 파일만 받아서 실행하면 됩니다

## 🛠️ 빌드 방법

### 방법 1: 배치 파일 사용 (가장 쉬움)
```
테스트모드_EXE_빌드.bat 더블클릭
```

### 방법 2: Python으로 직접 실행
```bash
python build_test_exe.py
```

## 📦 빌드 전 준비사항

1. Python 3.8 이상 설치되어 있어야 합니다
2. 필요한 패키지 설치:
   ```bash
   pip install -r requirements.txt
   ```

## ✅ 빌드 완료 후

빌드가 완료되면 다음 파일들이 생성됩니다:

1. **`dist/온카판_자동댓글_매크로_테스트.exe`**
   - 실제 실행 파일

2. **`타사용자_전달용/` 폴더**
   - exe 파일과 사용 안내 파일이 포함된 폴더
   - 이 폴더의 모든 파일을 압축하여 타사용자에게 전달하세요

## 📤 타사용자에게 전달하기

1. `타사용자_전달용` 폴더의 모든 파일을 선택
2. ZIP 파일로 압축
3. 타사용자에게 전달

타사용자는:
- 압축 해제 후 exe 파일만 더블클릭하면 됩니다
- Python 설치 불필요
- 추가 프로그램 설치 불필요

## ⚠️ 주의사항

- 이 exe는 **테스트 모드로만 실행**됩니다
- 실제 댓글은 작성되지 않으며, 시뮬레이션만 수행합니다
- 타사용자에게 이 점을 명확히 안내하세요

## 🔧 문제 해결

### 빌드 오류가 발생하면:

1. **PyInstaller 설치 확인**
   ```bash
   pip install pyinstaller
   ```

2. **모든 의존성 설치 확인**
   ```bash
   pip install -r requirements.txt
   ```

3. **Python 버전 확인**
   - Python 3.8 이상 권장
   - Python 3.13은 일부 호환성 문제가 있을 수 있음

4. **관리자 권한으로 실행**
   - 일부 경우 관리자 권한이 필요할 수 있습니다

### exe 파일이 실행되지 않으면:

- Windows Defender나 백신 프로그램이 차단했을 수 있습니다
- "추가 정보" → "실행"을 클릭하여 허용하세요
- 타사용자에게도 이 점을 안내하세요
