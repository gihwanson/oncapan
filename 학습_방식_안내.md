# 댓글 학습 방식 안내

## 현재 방식: **반자동**

### 1. 댓글 수집 (수동 실행 필요)
- **처음 한 번은 수동으로 실행해야 합니다**
- 방법 1: 웹 GUI에서 "댓글 수집하기" 버튼 클릭
- 방법 2: `댓글수집.bat` 파일 실행

### 2. 댓글 생성 시 학습 (자동)
- 매크로가 댓글을 생성할 때 **자동으로** 수집된 댓글을 로드하여 사용
- 별도 실행 불필요
- 수집된 댓글이 있으면 자동으로 학습에 활용

## 작동 방식

```
[수동] 댓글 수집 실행
   ↓
[자동] 댓글 데이터 저장
   ↓
[자동] 매크로 시작 시 댓글 로드
   ↓
[자동] 댓글 생성 시 학습 데이터 활용
```

## 자동화 옵션

### 옵션 1: 매크로 시작 시 자동 수집 (제안)
- 매크로 시작 시 댓글이 없으면 자동으로 수집
- 설정에서 켜고 끌 수 있음

### 옵션 2: 현재 방식 유지
- 처음 한 번만 수동으로 수집
- 이후 자동으로 활용
