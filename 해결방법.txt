========================================
Tkinter 오류 해결 방법
========================================

Python 3.13에서 tkinter가 제대로 설치되지 않은 경우가 있습니다.

해결 방법:

1. Python 재설치 (권장)
   - Python 공식 사이트에서 Python 3.11 또는 3.12 다운로드
   - 설치 시 "tcl/tk and IDLE" 옵션 선택
   - https://www.python.org/downloads/

2. tkinter 수동 설치
   - Python 3.13을 사용해야 하는 경우
   - 별도로 Tcl/Tk를 설치해야 할 수 있습니다

3. 다른 Python 버전 사용
   - Python 3.11 또는 3.12 사용 권장
   - tkinter가 더 안정적으로 작동합니다

========================================
임시 해결책
========================================

1. tkinter_설치확인.py 실행하여 경로 확인
2. 발견된 경로를 환경 변수로 설정
3. 또는 Python 재설치
