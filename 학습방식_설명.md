# 댓글 학습 방식 설명

## 현재 학습 방식

### 1. **Few-Shot Learning (Few-shot Learning)**
- 실제 수집된 댓글을 AI 프롬프트에 예시로 포함
- AI가 예시 댓글들을 보고 비슷한 스타일로 댓글 생성
- **25개의 실제 댓글 예시**를 매번 프롬프트에 포함

### 2. **유사 댓글 매칭**
- 게시글 내용과 유사한 키워드를 가진 댓글을 우선 선택
- 예: 게시글이 "배팅"에 관한 내용이면, "배팅" 관련 댓글을 예시로 사용
- 더 정확한 모방을 위해 유사한 맥락의 댓글 활용

### 3. **스타일 분석 활용**
- 수집된 댓글의 통계 분석 결과 활용
  - 평균 댓글 길이
  - 자주 사용되는 어미 (예: "~요", "~네요")
  - 댓글 길이 분포
- 이 정보를 AI에게 제공하여 실제 사용자 스타일 유지

### 4. **동적 학습**
- 댓글 수집 후 즉시 반영 (프로그램 재시작 불필요)
- 매번 최신 수집 데이터를 로드하여 사용
- 수집한 댓글이 많을수록 더 정확한 모방

## 작동 흐름

```
1. 댓글 수집
   ↓
2. 댓글 분석 (스타일, 패턴 추출)
   ↓
3. 게시글 읽기
   ↓
4. 유사한 댓글 25개 선택 (키워드 매칭)
   ↓
5. AI 프롬프트에 예시 포함
   ↓
6. AI가 예시를 보고 비슷한 스타일로 댓글 생성
```

## 개선 사항

### 이전 방식
- 댓글 예시 10-15개만 사용
- 단순히 예시 나열
- 게시글과 무관한 랜덤 댓글 사용

### 현재 방식 (개선됨)
- **댓글 예시 25개 사용** (더 많은 학습 데이터)
- **게시글과 유사한 댓글 우선 선택** (더 정확한 모방)
- **스타일 분석 결과 활용** (통계 기반 스타일 유지)
- **Few-shot learning 강화** (더 많은 예시로 더 정확한 모방)

## 효과

1. **자연스러움 향상**: 실제 사용자 댓글을 직접 모방하므로 더 자연스러움
2. **스타일 일관성**: 수집된 댓글의 스타일을 그대로 따름
3. **맥락 적합성**: 게시글과 유사한 댓글을 참고하므로 맥락에 맞는 댓글 생성
4. **AI 티 감소**: 실제 댓글 패턴을 학습하므로 AI처럼 보이지 않음

## 권장 사항

- **최소 50개 이상의 댓글 수집 권장** (더 다양한 스타일 학습)
- **정기적으로 댓글 수집** (최신 스타일 반영)
- **다양한 게시글에서 댓글 수집** (다양한 맥락 학습)
